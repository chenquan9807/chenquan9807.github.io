联想台式机启用tpm2.0

1、TPM2.0 已支持


此情况电脑可以直接升级win11

2、未发现TPM2.0

首先要确认自己的电脑时候支持TPM2.0

  • http://www.8fe.com/gongju/4218.html

可以从上面的连接查询自己的CPU是否支持。参考下面图片

否则正常正常升级渠道已经没了，歇歇吧（通过u盘，或者iso替换文件安装）

3、进入BIOS打开TPM2.0

  1. 首先在显示联想图标是按下F1（或者其他键）进入BIOS
  2. 找到“Secyrity”选项进入“TCG Security Setup”
  3. “Security Chip 2.0”改为“Enable”
  4. “F10”保存并退出
  5. 此时重启后就可以发现满足Win11升级条件了
